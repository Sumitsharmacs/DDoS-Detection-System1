DDoS Attack Detection System

Overview

This project is developed to detect Distributed Denial of Service (DDoS) attacks by monitoring network traffic in real time. The system captures packets, analyzes traffic patterns, and identifies abnormal traffic behavior based on predefined threshold values. When suspicious traffic exceeds the threshold, the system generates alerts through a live dashboard and administrator notification system.

Objectives----

Monitor network traffic continuously

Detect abnormal traffic patterns

Identify potential DDoS attacks

Provide real-time dashboard updates

Improve network security awareness

Key Features

Working Principle

The system continuously monitors incoming network packets and evaluates traffic volume. If the number of requests exceeds a predefined threshold, the system flags the traffic as suspicious and identifies it as a possible DDoS attack. Alerts are displayed on the dashboard, enabling administrators to monitor security status effectively.
